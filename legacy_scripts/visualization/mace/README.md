# MACE Visualization Legacy Scripts

Original MACE live plotting scripts copied here:

- `plot_mace_live.gp`
- `plotmace.sh`

Packaged command:

```bash
plotmace mace_train.log
plotmace mace_train.log 200 5
```

The installed command uses the packaged gnuplot script and no longer depends on `$HOME/scripts`.
